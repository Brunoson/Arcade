Flappy Bird

**(Documentation in progress)

Introduction

In this tutorial we attempt to create a computer program that learns to play the popular Flappy Bird game. This project utilizes the latest in reinforcement learning (deep Q-learning) and the Keras deep learning library.

(source: https://yanpanlau.github.io/2016/07/10/FlappyBird-Keras.html)

Why do we care?

This sort of research has implications far greater than just video games. Developing systems that autonomously learn representations of raw data (in this case pixel values) is extremely powerful and is particularly useful in the field of robotoics (self-driving cars), aviation, and practically any other field that is data-intensive.

Requirements

This tutorial assumes basic knowledge of neural networks, specifically convolutional neural networks as well as the deep Q-learning algorithm. For background on CNN's, check out this excellent blog post by Andrej Karpathy. I would also recommend taking a look at this tutorial by Michael Nielsen for a general overview of neural networks. For deep Q-learning, checkout DeepMind's paper.

Software:

  • Python 2.7, 3.5
  • Keras - available via pip or Github
  • Note that the Flappy Bird API is included in the repo. Source.

Files:

  • flappy.py

This project utilizes a "Locations.py" script (not included) that specifies the locations of various files/repositories on the local machine. Thus, you may need to make slight adjustments to the files in order for them to run. Specifically, the following locations are defined (among others potentially):

  • runs_dir = file path to saved model weights
  • arch_dir = file path to saved model architectures
